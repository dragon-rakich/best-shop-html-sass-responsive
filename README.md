# Best Shop - Responsive Landing Page

A mobile-first welcome page featuring a dynamic pricing calculator. Built to demonstrate modern layout techniques and interactive form handling.

## Features
* **Interactive Calculator:** Real-time price calculations using JavaScript.
* **Responsive Design:** Fully fluid layout using Flexbox and CSS Grid.
* **Optimized Build:** Bundled with Parcel for fast performance.

## Tech Stack
* **HTML5**
* **Sass** (Compiled via Parcel)
* **Vanilla JavaScript** (Logic for form calculations)
